# @arch-engine/governance-pack-authority

Architecture governance policy pack extending topology validation rules for authority boundaries.

## Installation
\`\`\`bash
npm install @arch-engine/governance-pack-authority@rc
\`\`\`

## Usage
This package registers transparently into the core reasoning pipeline if installed natively. 

## Optional Extensions
Governance packs and adapters are optional extensions. This pack requires the Arch-Engine CLI to perform execution and is not natively bundled.
