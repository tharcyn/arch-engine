# @arch-engine/core

Capability-driven architecture topology reasoning runtime.

## Installation
\`\`\`bash
npm install @arch-engine/core@rc
\`\`\`

## Usage
\`\`\`typescript
import { EngineRunner } from "@arch-engine/core"
\`\`\`

## Optional Extensions
Adapters and governance packs are optional extensions. Core provides the reasoning interface but does not extract topologies from native disk itself.
