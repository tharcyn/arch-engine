# @arch-engine/schema

Canonical schema contracts and shared types for Arch-Engine topology reasoning.

## Installation
\`\`\`bash
npm install @arch-engine/schema@rc
\`\`\`

## Usage
\`\`\`typescript
import type { EngineManifest } from "@arch-engine/schema"
\`\`\`

## Optional Extensions
Governance packs and adapters are optional extensions and evaluate these schemas upstream.
