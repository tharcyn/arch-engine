# Release Notes: v1.0.0-rc.1

## 1. Release Classification

**Release Candidate**

- Execution surface stable
- Identity surface stable
- Snapshot replay deterministic
- Registry federation neutral
- Policy-pack grammar stable

## 2. Stable Surfaces

The following `core` infrastructure surfaces are explicitly designated as stable for extraction:

- Execution topology evaluation
- `closureGraphHash` identity envelope
- Overlay merge algebra
- Authority-tier eligibility filtering
- Snapshot replay determinism
- Registry federation neutrality
- Policy-pack participation grammar

## 4. Included Example Artifacts

## 5. Non-Goals of RC

The following ecosystem properties are expressly uncoupled from the candidate and are not supported in this boundary:

- No registry service implementation
- No remote signing infrastructure
- No hosted federation mirror
- No adapter marketplace
