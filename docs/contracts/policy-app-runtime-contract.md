# Policy App Runtime Contract

The Arch-Engine Policy App Execution Model expands single policy-packs into full-fledged stateful governance applications.

## Declarations

Arch-Engine supports:
1. **Automation Bundles**: Bundling multi-stage triggers, findings, and operator dependencies into cohesive `PolicyAppManifest` deployables.
2. **Registry Sync Apps**: Automating multi-tenant registry synchronization.
