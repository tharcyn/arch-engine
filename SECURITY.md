# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| 0.1.x   | :x:                |

## Reporting a Vulnerability

Please report security issues to security@arch-engine.dev.

Do not open a public issue for security vulnerabilities. We will acknowledge receipt within 48 hours and provide a timeline for resolution.
