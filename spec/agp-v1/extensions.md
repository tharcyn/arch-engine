# AGP v1 Extension Lifecycle Specification

## Overview
The Extension Lifecycle defines the safe evolution model of the AGP specification.

## Extension Pipeline
1. **Proposal Lifecycle**: Initial submission.
2. **Review Lifecycle**: Security and determinism auditing.
3. **Ratification Lifecycle**: Acceptance into the canonical standard.
4. **Experimental Classification**: Pre-production capabilities.
5. **Stable Classification**: Fully supported and backwards-compatible capabilities.
6. **Deprecation Lifecycle**: Graceful removal of legacy schemas.
7. **Compatibility Guarantees Across Revisions**: How older capsules run on newer extensions.
