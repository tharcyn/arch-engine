# AGP v1 Node Topology Specification

## Overview
This defines the infrastructure roles for AGP compliance, dictating the different node surfaces that can be operated.

## Node Types
1. **Reference Node Behavior**: The canonical full AGP v1 participant.
2. **Observer Node Responsibilities**: Read-only, verification, and audit execution.
3. **Certification Node Responsibilities**: Ecosystem adapter badging and validation loops.
4. **Federation Bootstrap Node Responsibilities**: Seeding new multi-organization trust networks.
5. **Verification-Only Node Guarantees**: Restricted capabilities for restricted environments.
6. **Deployment Topology Expectations**: Horizontal scaling limits and data sovereignty bounds.
